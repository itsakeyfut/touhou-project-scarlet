## 確定した設計方針

| 項目 | 決定内容 |
|------|---------|
| **スコープ** | Stage 1〜6 + Extra（フランドール）+ Phantasm（独自） |
| **自機** | 霊夢・魔理沙（原作）、咲夜（独自追加、Phase 20） |
| **アートスタイル** | モダンHDピクセルアート（Shovel Knight / Celeste レベル） |
| **解像度** | 16:9ウィンドウ（1280×720基準）、4:3プレイエリア（384×448）中央配置 |
| **BGM** | 3モード（原作アレンジ / チップチューン / オリジナル）切り替え可 |
| **難易度** | Easy / Normal / Hard / Lunatic / Extra / Phantasm |
| **プラットフォーム** | Windows + Linux（Steam Deck含む）+ Steam配信 |
| **言語** | 日本語・英語対応 |
| **入力** | キーボード（設定可）+ ゲームパッド（XInput/DInput） |

## 開発方針

- **商業品質目標**: 非営利ではあるが Steam リリースを想定したクオリティを追求
- **現代的ピクセルアートリメイク**: WGSLシェーダーによる美麗な演出を重視
- **忠実ベース＋QoL**: ゲームプレイはEoSD準拠、ヒットボックス表示・リプレイ等の現代機能を追加
- Phase 4〜9 からシェーダーを段階的に導入、ゲームプレイと並行して磨く
- 各フェーズは単独で動作確認できる単位
- `just check` (clippy + fmt) が常に通る状態を維持
- セーブデータはJSONで永続化（`serde_json`使用）
- BGM/SFX の著作権注意: 開発中はダミー音源を使用すること
- シェーダーはホットリロード (`file_watcher` feature) で即座に確認
