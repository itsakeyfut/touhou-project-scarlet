## バランス調整の基準

### 難易度カーブ

```
Easy:     初心者が1時間程度で Stage 3 到達できる
Normal:   弾幕STG経験者が数時間練習でクリアできる
Hard:     上級者向け、Normalクリア後しばらく詰まるレベル
Lunatic:  熟練弾幕プレイヤー向け、かなりの練習が必要
Extra:    Lunaticクリア相当の技術が必要
Phantasm: プロジェクト最高難易度、完全習熟者向け
```

### スペルカードキャプチャ率（目標）

| 難易度 | 目標キャプチャ率（中級者） |
|-------|----------------------|
| Easy | 90%以上 |
| Normal | 70%以上 |
| Hard | 40%以上 |
| Lunatic | 20%以上 |
| Extra | 10%以上 |
| Phantasm | 5%以上 |
