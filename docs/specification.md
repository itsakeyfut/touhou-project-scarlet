## 1. プロジェクトビジョン

### コンセプト

商業リリース品質を目指した**現代的2Dピクセルアートリメイク**。東方紅魔郷の弾幕STGとしての本質を忠実に再現しつつ、WGSLシェーダーによる美麗な演出とモダンHDピクセルアートで新たな体験を提供する。

### 設計原則

| 原則 | 内容 |
|------|------|
| **ゲームプレイ忠実性** | 弾速・当たり判定・スコア計算はEoSD準拠 |
| **現代的QoL** | ヒットボックス表示・リプレイ・実績など現代STGの標準機能を追加 |
| **ビジュアル革新** | モダンHDピクセルアート＋WGSLグロー/ブルームシェーダー |
| **完全スコープ** | Stage 1〜6 + Extra + Phantasm（全ボス・全スペルカード） |
| **商業品質** | Steamリリースを想定した完成度（ただし非営利） |

### 確定した主要設計決定

| 項目 | 決定内容 |
|------|---------|
| スコープ | Stage 1〜6 + Extra（フランドール）+ Phantasm 難易度 |
| 自機 | 霊夢・魔理沙（原作）、咲夜（独自追加）、将来的に追加キャラ |
| 再現度 | EoSD忠実ベース + QoL改善（ヒットボックス表示等） |
| アートスタイル | モダンHDピクセルアート（Shovel Knight / Celeste レベル） |
| 解像度 | 16:9ウィンドウ（1280×720〜スケーラブル）、4:3プレイエリア中央配置 |
| BGM | 複数モード（原作アレンジ / チップチューン / オリジナル） |
| 追加モード | スペルプラクティス、リプレイ、スペルカードコレクション、ミュージックルーム、実績 |
| プラットフォーム | Windows、Linux（Steam Deck含む）、Steam配信 |
| 難易度 | Easy / Normal / Hard / Lunatic / Extra + Phantasm |
| ストーリー | 立ち絵＋テキストダイアログ、日本語・英語対応 |
| 入力 | キーボード（設定可）＋ゲームパッド（XInput/DInput） |
| QoL | ヒットボックス表示、グレイズ圏可視化、FPS表示、ウィンドウサイズ変更 |

## 4. ゲームフロー

### 4.1 メインフロー

```
タイトル画面（言語選択ボタン）
    ↓
メインメニュー
    ├──→ ゲームスタート
    │        ↓
    │    キャラクター選択（霊夢 / 魔理沙 / 咲夜）
    │        ↓
    │    ショットタイプ選択（A / B）
    │        ↓
    │    難易度選択（Easy / Normal / Hard / Lunatic）
    │        ↓
    │    Stage 1〜6 → ステージクリア → エンディング
    │        ↓（Normal以上・ノーコンティニューでクリア）
    │    Extra Stage 解放
    │        ↓（全キャラ×全タイプでExtraクリア）
    │    Phantasm 解放
    │
    ├──→ スペルプラクティス
    ├──→ リプレイ再生
    ├──→ スペルカードコレクション
    ├──→ ミュージックルーム
    ├──→ 実績
    ├──→ スコアランキング
    └──→ 設定（BGMモード / キー設定 / 画面設定 / 言語）
```

### 4.2 プレイ中フロー

```
ステージ開始
    ↓
背景スクロール＋ザコ敵 Wave 1〜N
    ↓（存在する場合）
中ボス戦
    ↓
ザコ敵 Wave（中ボス後）
    ↓
ボス登場（スクロール停止）
    ├──→ ノンスペル（通常攻撃フェーズ）
    └──→ スペルカード（固有弾幕フェーズ）
    ↓（全フェーズ撃破 or タイムアウト）
ステージクリアボーナス計算・表示
    ↓
次ステージへ（または エンディング）
```

### 4.3 ゲームオーバー・コンティニュー

- **ゲームオーバー条件**：残機 0 で被弾
- **コンティニュー**：残機3・ボム3の状態で現ステージ再スタート
  - コンティニュー後はハイスコアランキングに登録されない
  - Extra / Phantasm はコンティニュー不可
- **クリア種別**：

| 種別 | 条件 | 結果 |
|------|------|------|
| ノーマルエンド | コンティニューあり6ステージクリア | 通常エンディング |
| グッドエンド | Normal以上ノーコンティニューでクリア | Extraステージ解放 |
| Extraエンド | Extraステージクリア | Extra専用エンディング |
| Phantasmエンド | Phantasmクリア | 隠しエンディング |

## 5. プレイヤーキャラクター

### 5.1 共通パラメータ

| パラメータ | 値 |
|-----------|-----|
| ヒットボックス半径 | 2.5 px（視覚的には小さな点として表示） |
| グレイズ半径 | 16 px |
| 通常移動速度 | 4.5 px/frame（270 px/秒） |
| 低速移動速度（フォーカス） | 2.0 px/frame（120 px/秒） |
| 初期残機 | 3 |
| 初期ボム数 | 3 |
| 最大ボム | 8 |
| 無敵時間（被弾後） | 3秒（180フレーム） |

### 5.2 霊夢・博麗（Reimu Hakurei）

**ショットタイプA：夢想**
- ショット：霊力弾（広角・扇状拡散）、ホームミサイル（自動誘導）
- フォーカス：集中ショット（前方3本）
- ボム：「夢想天生」（全方位霊力展開・約3.5秒）

**ショットタイプB：風神**
- ショット：説得の針（前方集中・高速ニードル）
- フォーカス：3列の密集ニードル
- ボム：「天覇風神脚」（衝撃波を前方に放つ）

**特性**：バランス型。どちらのタイプも初心者〜中級者向け。

### 5.3 霧雨・魔理沙（Marisa Kirisame）

**ショットタイプA：恋符**
- ショット：魔力弾（前方集中・高威力）
- フォーカス：5-wayレーザービーム
- ボム：「恋符「マスタースパーク」」（極太レーザー・高ダメージ）

**ショットタイプB：星符**
- ショット：星型弾（斜め方向にも広がる）
- フォーカス：誘導小弾群
- ボム：「彗星「ブレイジングスター」」（体当たり突進・完全無敵）

**特性**：高威力・高リスク型。スコアを稼ぎやすいが立ち回りが難しい。

### 5.4 十六夜 咲夜（Sakuya Izayoi）— 独自追加

**ショットタイプA：銀時計**
- ショット：ナイフ（前方・高速・高頻度）
- フォーカス：扇状ナイフ投擲（広角）
- ボム：「時間停止「パーフェクトスクウェア」」（全弾停止・ナイフに変換→一斉再開）
- **固有能力**：フォーカス中に時折弾がスローモーションになる（視覚エフェクトのみ、ゲームロジックには影響しない）

**ショットタイプB：幻時計**
- ショット：幻影ナイフ（分裂弾）
- フォーカス：前方集中の直線ナイフ
- ボム：「時間消去「タイムエクスプロージョン」」（画面全体に時計の針をまき散らす）

**特性**：独自アレンジキャラ。中〜上級者向けの高火力スタイル。

**注記**：咲夜はStage 5のボスキャラクターでもあるが、プレイアブルキャラクターとしての性能はボスとは独立して設計する。

### 5.5 将来のプレイアブルキャラクター

以下のキャラクターをPhantasm解放後の隠しキャラとして実装予定（要検討）：

- **パチュリー・ノーレッジ** — 魔法使い、5属性魔法弾

## 6. ゲームシステム

### 6.1 ショットシステム

- Zキー（長押し連射）でショット
- フォーカス時（Shiftキー）はパターンが変化（集中 or 別形態）
- パワーレベルに応じてショット数・威力が8段階で上昇

### 6.2 パワーシステム

| パワー値 | 段階 |
|---------|------|
| 0〜7 | Lv1 |
| 8〜15 | Lv2 |
| 16〜31 | Lv3 |
| 32〜47 | Lv4 |
| 48〜63 | Lv5 |
| 64〜79 | Lv6 |
| 80〜95 | Lv7 |
| 96〜127 | Lv8 |
| 128 | MAX |

- パワーアイテム（小）：+1
- パワーアイテム（大）：+10
- 被弾時：パワーを一定量失う
- MAXパワー時、画面最上部に移動すると全アイテムを自動収集

### 6.3 ボムシステム（スペルカード）

- **入力**：Xキー
- **最大ストック**：8個
- **効果**：
  - キャラ固有の攻撃を展開（全敵にダメージ）
  - 範囲内の弾を全消去 → スター（★）アイテムに変換
  - 画面内の全アイテムを自動収集
  - 発動中・発動後しばらく無敵
- **カウンターボム**：被弾直後（約0.1秒以内）にボムを使うと被弾を無効化（ボム1消費）

### 6.4 グレイズシステム

- 敵弾がヒットボックス半径（2.5px）外かつグレイズ半径（16px）内を通過すると発生
- 1グレイズ = スコア +500点 + グレイズカウンター +1
- 同一弾は1回のみグレイズ可能（レーザーは継続的にグレイズ）
- グレイズカウンターは累積（最大9999）

### 6.5 残機・エクステンドシステム

| 条件 | 効果 |
|------|------|
| 被弾（ヒットボックスに弾が当たる） | ミス → 残機 -1 |
| 残機 0 で被弾 | ゲームオーバー |
| スコア 10,000,000点 | 残機 +1 |
| スコア 20,000,000点 | 残機 +1 |
| スコア 40,000,000点 | 残機 +1 |
| スコア 60,000,000点 | 残機 +1 |
| 1UPアイテム取得 | 残機 +1 |

## 8. ボスキャラクター

### 8.1 ボス共通仕様

- **フェーズ構成**：ノンスペル（通常攻撃）＋ スペルカード を複数フェーズ
- **HPゲージ**：フェーズごとにセグメント表示（右HUDパネル）
- **タイムリミット**：スペルカードごとに設定（制限時間内に撃破できなければ次フェーズへ）
- **スペルカードキャプチャ**：ノーミス・ノーボムで撃破 → スペルカードコレクションに記録、ボーナス加算

## 13. 追加モード

### 13.1 スペルプラクティスモード

- **解放条件**：スペルカードを1枚でも初回キャプチャ（ノーミス・ノーボム撃破）
- **機能**：
  - 解放済みスペルカードを自由に選択して繰り返し練習
  - 自機・難易度は自由に設定可能
  - 挑戦回数・成功回数を記録
  - スコアはメインスコアには影響しない

### 13.2 リプレイシステム

- **自動保存**：クリア時・ゲームオーバー時に最新リプレイを1件自動保存
- **手動保存**：ゲームオーバー画面・クリア画面から手動保存（最大10件）
- **リプレイ内容**：入力データ（フレームごとのキー入力）のみ記録し、シード値で再現
- **リプレイ再生**：メインメニューから選択、早送り（×2、×4）、一時停止対応
- **ファイル形式**：`.scarlet-replay`（バイナリ / serde シリアライズ）

### 13.3 スペルカードコレクション

全スペルカードの取得状況を記録・表示：

| 列 | 内容 |
|----|------|
| キャラクター×タイプ | 霊夢A/B、魔理沙A/B、咲夜A/B |
| 難易度 | Easy / Normal / Hard / Lunatic |
| 取得状況 | ◎ノーミスノーボム / ○撃破 / ×未撃破 |
| チャレンジ回数 | 挑戦回数 |

- 全スペルカードを◎取得で称号（実績）解放
- ステージ別・ボス別のフィルタリング対応

### 13.4 ミュージックルーム

- ゲーム内でアンロックしたBGMを自由に試聴
- 解放条件：対応するステージをプレイすることで解放
- BGMモード切り替え（原作アレンジ / チップチューン / オリジナル）
- 楽曲情報（タイトル・作曲クレジット）を表示

### 13.5 実績（Achievement）システム

- Steam実績 + ゲーム内実績の二重管理
- Steam未接続環境ではゲーム内のみ記録

**実績カテゴリ**：

| カテゴリ | 例 |
|---------|-----|
| ストーリー | 各ステージ初クリア、ノーコンティニュークリア |
| スペルカード | 各スペルカード初キャプチャ、全スペルカードキャプチャ |
| スコア | スコア閾値達成（1000万・5000万・1億点） |
| 技術 | ノーミスクリア、ノーボムクリア、Phantasmクリア |
| コレクション | 全BGM解放、全スペルカードコレクション完成 |

### 18.3 ウィンドウサイズ・フルスクリーン

- 設定から切り替え可能（ゲーム中も変更可）
- ウィンドウサイズ：2× / 3× / 4× / フルスクリーン
- フルスクリーン時はモニター解像度に合わせてピクセルパーフェクトスケール

### 18.4 FPS表示

- デバッグ兼パフォーマンス確認用
- 設定からオン/オフ（デフォルトはオフ）
- 画面左上隅にオーバーレイ表示

- SEボリューム（0〜100%）

## 20. ビジュアルスタイル方針

### 20.1 アートスタイル

**モダンHDピクセルアート**（Shovel Knight / Celeste レベル）

- 高解像度ドット絵（アンチエイリアスなし、ニアレストネイバー拡大）
- 豊かなカラーパレット（制限なし）
- キャラクター・背景・UIすべてピクセルアート統一

| プレイヤー（咲夜） | 32×48 px | 8〜12フレーム |
| ボス | 64×64 px | 4〜8フレーム |

### 20.4 HDR・ブルーム設定

- カメラはHDRモード（`Camera { hdr: true }`）
- 弾の `glow_color` に `LinearRgba` 値 > 1.0 を設定することでブルームが発光
- 通常スプライトは値 ≤ 1.0（ブルームなし）
- `ImagePlugin::default_nearest()` でピクセルアートのニアレストネイバーフィルタリング
