# 東方紅魔郷

Touhou Koumakyou — Embodiment of Scarlet Devil の非公式 Rust / Bevy 実装。

## 操作方法

### ゲームプレイ中

| キー | 操作 | 状態 |
|------|------|------|
| `←` / `A` | 左移動 | 実装済み |
| `→` / `D` | 右移動 | 実装済み |
| `↑` / `W` | 上移動 | 実装済み |
| `↓` / `S` | 下移動 | 実装済み |
| `Shift` (左/右) | 低速移動（フォーカス） | 実装済み |
| `Z` | ショット | 実装済み |
| `X` | ボム | 未実装 |
| `Esc` | ポーズ | 未実装 |

### メニュー / 画面遷移

| キー | 操作 | 状態 |
|------|------|------|
| `←` / `→` | 項目選択 | 未実装 |
| `Z` / `Enter` | 決定 | 未実装 |
| `X` / `Esc` | キャンセル・戻る | 未実装 |

## 開発用コマンド

```bash
# デバッグモードで起動（プレイ画面に直接遷移・プレイエリア境界表示）
just dev

# 通常起動（タイトル画面から）
just run

# テスト
just test

# ビルド
just build
```
